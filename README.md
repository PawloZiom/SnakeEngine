# 🐍 Snake Engine
Snake Engine is a hobby project written in Python.
> ⚠️ Snake Engine is currently under active development and the API may change frequently.

## 🚀 Features
- [x] Basic OpenGL-based renderer
- [x] Basic ECS architecture
- [x] UI system (Unity UI-like)
- [x] Input system
- [x] Basic 3D directional audio system
- [ ] Asset Management (Caching & Asynchronous loading)
- [ ] Physics (powered by Bullet)
- [ ] Integrated Editor
- [ ] Packaging tools

## 🔨 Building
Currently, the engine is **not** designed to be built into a standalone library or executable. It is intended to be used as part of your project's source tree. I am planning to add proper build support and packaging scripts in the future, but for now, please include the engine source files directly within your project directory or try experimenting with PyInstaller.
